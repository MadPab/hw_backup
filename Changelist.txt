Текущие изменения:

1. Теперь целевой профиль и количество загружаемых фото программа будет запрашивать у пользователя. Ппользователь может вводить как id, так и screen_name
2. Файлы больше не загружают фото локально перед их загрузкой в облачное хранилище
3. Исправлена ошибка, когда вся логика была в одном классе и использовались глобальные переменные. Теперь основная логика программы разбита на два класса и метод для записи информации в файл. 
4. Исправлена ситуация, когда количество лайков у фото равно. Теперь при одинаковом количестве лайков добавляется текущая дата
5. Исправлен файл зависимостей, теперь в нем указаны только используемые в проекте библиотеки (использована команда python -m pipreqs.pipreqs <path>, так pipreqs <path> не выполнялась)
6. Исправлена ситуация с сохранением в json-файл. Теперь он является отдельной функцией, которая принимает на вход параметры и может быть вызвана при необходимости
7. Исправлена ошибка с токенами, теперь они находятся в специальном файле settings.ini

Будущие изменения:

1. Добавить возможность загружать фото\видео на Google.Drive
2. Добавить возможность сохранять фотографии из других альбомов
